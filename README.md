# zzHouse
# 郑州买房，将数据进行可视化展示。

## 演示

- 附近美食、超市便利店、学校

  - http://lifemap.misteryliu.com/ 

- 附近房子每天价格和数量变动

  - http://house.misteryliu.com/   

- 工作分布

  - http://workmap.misteryliu.com/ 

- 附近楼盘分布
  
  - http://housemap.misteryliu.com/ 

## 粗略教程

- 项目中每个文件夹都是一个小项目
- 以map结尾的项目，需要注册百度的apik，在index.html中
- house项目需要填写数据库账号密码
- map项目不能在本地运行，需要部署在服务器上
